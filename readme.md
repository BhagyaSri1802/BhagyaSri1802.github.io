Technical Requirements
HTML Structure
Single-Page Layout: Build a one-page site using HTML.
Content Sections: Use at least two <div> tags to create distinct sections of content.
SEO Metadata: Include a <meta name="description"> tag to describe the site for search engines.
Media: Include at least one image.
Hyperlinks: Use an <a> tag to link to an external website.

CSS Styling
External Stylesheet: All styles must be contained in a separate stylesheet.css file.
    Selectors:
    Tag Selector: Style at least one HTML tag directly (e.g., p, h1).
    Class Selector: Create one class and apply it to at least two different elements.
    ID Selector: Create one unique ID and apply it to a single element.
    Interactivity: Use a CSS hover selector to change the color of links when the cursor moves over them.
    Box Model: Modify at least two box-model properties (e.g., padding, margin, border, height, or width) on any element.

Deployment & Workflow
Version Control: Host the code in a GitHub repository.
Live Deployment: Deploy the site using GitHub Pages so it is accessible at https://<your-username>.github.io.
